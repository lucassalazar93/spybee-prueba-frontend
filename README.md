# Spybee – Prueba Técnica Frontend

Este repositorio contiene el desarrollo de una **prueba técnica de frontend**, cuyo objetivo es construir un dashboard funcional para la gestión y visualización de proyectos, priorizando una buena experiencia de usuario, claridad visual y una estructura de código mantenible.

La aplicación simula un escenario real con listado de proyectos, filtros, métricas y visualización geográfica, siguiendo buenas prácticas de desarrollo frontend moderno.

---

## Tecnologías utilizadas

- **Next.js** (App Router)
- **TypeScript**
- **CSS Modules**
- **Mapbox GL JS**
- **Imports dinámicos** para componentes dependientes del navegador
- **Estado en cliente** organizado por funcionalidad

No se utilizaron librerías externas de UI. Todo el layout y los estilos fueron implementados de forma manual.
